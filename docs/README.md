<p align="center">
	Catppuccin主题文档——为活力四射的你准备的舒缓柔美配色方案！
</p>

<p align="center">
Catppuccin是一个社区驱动的柔美主题，旨在成为低对比度与高对比度主题之间的完美平衡。它包含4种令人舒缓的温暖风味，每种风味包含26种赏心悦目的颜色，非常适合编程、设计等多种场景！此外，本仓库还追踪实际调色板的开发进度，包含组织级资源、素材以及为维护者/开发者提供的代码示例。
</p>
